# Chess Console

A console-based chess game developed in C# that implements the complete logic of a traditional chess match.

This project was created to practice object-oriented programming concepts, inheritance, polymorphism, encapsulation, and game rule implementation.

## Features

* Complete chess board representation
* Piece movement validation
* Turn-based gameplay
* Capture mechanics
* Check detection
* Checkmate detection
* Console interface
* Object-oriented architecture

## Technologies

* C#
* .NET Framework
* Console Application

## Running the Project

Clone the repository:

```bash
git clone https://github.com/ocainadev/chess-console.git
```

Open the solution file:

```text
xadrez_console.sln
```

Build and run the project using Visual Studio.
